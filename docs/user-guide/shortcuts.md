## Customization

### Custom Shortcuts
You can customize keyboard shortcuts in Settings:

1. Open Settings (`Ctrl+,` / `Cmd+,`)
2. Go to **Keyboard Shortcuts**
3. Click on any shortcut to modify
4. Press new key combination
5. Save changes

### Shortcut Conflicts
- NoteSage will warn about conflicts with system shortcuts
- Some shortcuts may vary by operating system
- Browser-based shortcuts take precedence in web version

## Tips for Efficiency

### Workflow Shortcuts
1. **Quick Note Creation**: `Ctrl+N` → Type title → Start writing
2. **Fast Navigation**: `Ctrl+K` → Type note name → Enter
3. **Rapid Formatting**: Select text → `Ctrl+B` for bold
4. **List Management**: `Tab`/`Shift+Tab` to indent/outdent
5. **Block Operations**: `Ctrl+Shift+Up/Down` to reorganize

### Memory Aids
- Most shortcuts follow standard conventions (Ctrl+C for copy)
- Formatting shortcuts match common editors
- Slash commands are discoverable by typing `/`
- Context menus show available shortcuts

### Practice Suggestions
1. Start with basic navigation shortcuts
2. Learn text formatting shortcuts you use most
3. Master list and heading shortcuts for structure
4. Explore slash commands for content insertion
5. Customize shortcuts for your workflow

Remember: Shortcuts may vary slightly between operating systems. This guide shows Windows/Linux shortcuts first, followed by macOS equivalents in parentheses.
